# Phonebook Application

Welcome to the Phonebook Application! This program allows users to create, show, search, update, and delete contacts. 

## Features

1. **Create Contact**: Add a new contact with a first name, last name, phone number, and email.
2. **Show Contacts**: Display all contacts stored in the phonebook.
3. **Search Contact**: Find a contact by name.
4. **Delete Contact**: Remove a contact by name.
5. **Update Contact**: Update the details of an existing contact.
6. **Exit**: Exit the phonebook application.

## Getting Started

### Prerequisites

- Python 3.x
- `re` module (usually included in Python standard library)

### Running the Application

1. Clone the repository or download the code.
2. Ensure you have Python installed on your system.
3. Open a terminal or command prompt.
4. Navigate to the directory containing the phonebook application.
5. Run the following command:

```bash
python phonebook.py
```
